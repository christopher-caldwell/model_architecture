## Architectural Notes

Bootstrap is the composition root. It wires concrete adapters to application services, but contains no business logic.

The member identifier generator is implemented here as `RandomIdentGenerator` using `node:crypto`. This avoids adding a `nanoid` dependency while preserving the Rust idea: application depends on an `IdentGenerator` port, and the edge supplies the implementation.

`server-bootstrap` re-exports command input types for transport convenience without exposing domain internals as transport DTOs.
