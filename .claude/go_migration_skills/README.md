# Go Migration Skills

This package contains skill folders for migrating the Rust modular architecture demo into idiomatic Go while preserving the same behavior, contracts, and boundaries.

## Use order

Start with the inventory and language-neutral spec. Then create layout and boundary rules. Then implement domain, ports, UOW, application, persistence, bootstrap, transports, examples, tests, vertical slices, and architecture review.

## Core principle

Same architecture and behavior. Idiomatic Go implementation. No hallucinated replacement architecture.
